🎨 Moodboard Studio Pro
Project Overview
Moodboard Studio Pro is a modern, real-time interior design moodboard application that allows users to create, organize, and visualize their design ideas. Built with Supabase as the backend, this application provides a seamless experience for interior designers, homeowners, and creative professionals to collect and arrange design elements like furniture, colors, lighting, decor, and plants into beautiful moodboards.

🚀 Features
1. User Authentication
Secure signup and login system using Supabase Auth

Personalized user accounts with private rooms and moodboards

Session management with real-time authentication state

2. Room Management
Create multiple design rooms (e.g., Bedroom, Living Room, Home Office)

Edit and delete rooms as needed

Each room acts as a separate moodboard canvas

3. Visual Moodboard Creation
Add design elements with titles, descriptions, and colors

Upload or paste image URLs for visual representation

Drag-and-drop interface to rearrange elements

Real-time updates across all devices

4. Element Library
Pre-designed elements for quick inspiration:

🪑 Furniture (sofas, chairs, tables, beds)

🎨 Colors (curated color palettes)

💡 Lighting (lamps, pendants, chandeliers)

✨ Decor (art, mirrors, vases, pillows)

🌿 Plants (indoor plants and greenery)

5. AI-Powered Image Search
Search for any design element (e.g., "cozy mat", "modern sofa")

Real images fetched from Unsplash

One-click add to moodboard

Smart suggestions based on search terms

6. Quick Templates
Pre-made room designs for instant inspiration:

🏢 Modern Living Room

🌿 Bohemian Paradise

⬜ Minimalist Space

❄️ Scandinavian Retreat

7. Color Palette Generator
Generate random color schemes

One-click to add colors to your moodboard

Visual color swatches for easy selection

8. Export & Share
Export moodboard as PNG image

Shareable links for collaboration

Copy link to clipboard functionality

9. Real-Time Database
Powered by Supabase PostgreSQL

Instant synchronization across devices

Cloud-based storage for all user data

10. Responsive Design
Works on desktop, tablet, and mobile devices

Clean, modern interface with smooth animations
